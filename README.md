# 🚀 Curso AI-900 - Fundamentos de Inteligência Artificial

Este projeto contém a estrutura de um site simples desenvolvido com **HTML + Bootstrap**, com o objetivo de organizar os conteúdos do curso **AI-900 (Microsoft Azure AI Fundamentals)**.

## 📚 Estrutura do Projeto

O site é composto por uma página principal e páginas individuais para cada dia do curso:

```
/
├── index.html
├── dia01.html
├── dia02.html
├── dia03.html
├── dia04.html
├── dia05.html
└── README.md
```

## 🌐 Página Principal

A página `index.html` funciona como um menu central, contendo links para cada dia do curso:

* 📅 Dia 1 – Introdução à IA
* 📅 Dia 2 – Machine Learning
* 📅 Dia 3 – Visão Computacional
* 📅 Dia 4 – Processamento de Linguagem Natural
* 📅 Dia 5 – IA Generativa

## 🎨 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5 (via CDN)

## ▶️ Como Executar o Projeto

1. Baixe ou clone este repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra a pasta do projeto

3. Execute o arquivo `index.html` no navegador

## 🌍 Publicação (GitHub Pages)

Para publicar o projeto online:

1. Suba os arquivos para o GitHub
2. Vá em **Settings > Pages**
3. Selecione a branch `main`
4. Clique em **Save**

Após isso, seu site estará disponível em:

```
https://seu-usuario.github.io/seu-repositorio
```
