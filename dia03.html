Revisão: Machine Learning

Machine Learning é quando treinamos um modelo com dados para encontrar padrões.

Fluxo básico:

  • Você fornece dados
  • O modelo aprende padrões
  • Ele faz previsões

Importante:

O modelo não entende… ele aprende padrões matemáticos.

LLMs (Modelos de Linguagem)

LLMs são modelos treinados com enormes volumes de texto.

Como eles funcionam:

Eles preveem qual é a próxima palavra mais provável.

Exemplo:

"O céu é..." → azul

Importante:

Ele não sabe o que é o céu… ele só prevê.

ATIVIDADE: Testar no ChatGPT: 1. Pergunta confusa 2. Pergunta difícil 3. Informação errada Refletir: Ele erra? Ele parece confiante?

Tokens

O modelo não lê palavras inteiras.

Ele divide o texto em pedaços chamados tokens.

Por que isso importa?

  • Cada token tem custo
  • Existe limite de tokens
  • Respostas grandes consomem mais

Pergunta:

Por que respostas maiores custam mais?

Transformer

É a arquitetura que permite que a IA funcione tão bem.

Antes:

  • Modelos liam palavra por palavra

Agora:

  • Lê tudo ao mesmo tempo
  • Entende contexto completo

Resultado:

Mais rapidez + mais precisão

Mecanismo de Atenção

Permite que o modelo foque nas partes importantes da frase.

Exemplo:

"O aluno estudou porque ele queria passar"

A IA entende que "ele" = aluno

Isso é o que dá contexto para a IA.

Engenharia de Prompt

Prompt é como você conversa com a IA.

Estrutura ideal:

Contexto
Tarefa
Formato
Restrições

Regra principal:

Quanto mais claro você for, melhor será a resposta.

ATIVIDADE: Prompt ruim: "Explique IA" Melhorar usando: - contexto - objetivo - formato Comparar resultados

Técnicas de Prompt

  • Zero-shot → sem exemplo - ex: "Classifique este sentimento: 'Estou muito feliz hoje"
  • Few-shot → com exemplo - ex: Classifique os sentimentos: estou triste = negativo. hoje foi incrível = positivo. Agora classifique "estou muito feliz hoje"
  • Role prompt → dar papel para IA. ex: você é um professor de ensino médio. Explque o que é IA.

Importante:

Você está ensinando a IA como responder.

ATIVIDADE: Criar um assistente: - RH - Marketing - Atendimento Entregar: Prompt + resultado

Temperatura

Controla o nível de criatividade da IA.

  • Baixa → resposta mais precisa
  • Alta → resposta mais criativa

Uso:

  • Relatórios → baixa
  • Ideias → alta

Top-P

Controla a variedade das palavras escolhidas.

Diferença:

  • Temperatura → aleatoriedade
  • Top-P → diversidade

Alucinação

A IA pode inventar respostas incorretas.

Problema:

  • Parece confiável
  • Nem sempre está certa

Regra:

Sempre validar informação importante

RAG (Geração aumentada por recuperação)

Combina IA com dados reais.

Como funciona:

  • Busca dados
  • Envia para IA
  • IA responde com base nisso
